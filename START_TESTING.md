# Start Testing Developer Atlas

Thank you for testing Developer Atlas.

You are not being tested.  
The pack is being tested.

## Step 1 — Choose one path

Pick the path that best matches you:

- New to web development → `packs/beginner-preview/`
- Have a basic Vue project → `packs/rookie/`
- Want programming terms explained → `packs/library-preview/`
- Use AI coding tools → `packs/ai-collaboration-preview/`

Do not test everything at once.

## Step 2 — Open the pack

Each pack folder has its own `README.md`.

Start there.

The Rookie Pack may include:

```txt
START_HERE.md
mission-dashboard.html
developer-atlas-rookie-pack.zip
```

## Step 3 — If there is a zip

1. Download or copy the zip.
2. Extract it.
3. Open the extracted folder.
4. Start with `START_HERE.md` or `mission-dashboard.html`.

## Windows browser tip

Sometimes `.html` files open in VS Code.

If that happens:

1. Open the extracted folder in Windows File Explorer.
2. Right-click `mission-dashboard.html`.
3. Choose **Open with**.
4. Pick Chrome, Edge, Firefox, or your browser.

## Step 4 — Stop when confused

Please stop and write down:

- what you clicked
- what you expected
- what confused you
- what you wanted next

Confusion is useful feedback.
