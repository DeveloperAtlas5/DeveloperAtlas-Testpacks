# Developer Atlas Test Packs

This repository contains clean tester-facing Developer Atlas packs.

Developer Atlas helps learners, builders, and AI work from the same programming map through guided missions, concept libraries, project start guides, and AI coding standards.

## Packs

- **Rookie Pack** — a small first-mission reader for someone with an existing Vue to-do project.
- **Full Pack** — missions, concept library, project start guides, AI guidance, and reviewer material. (under construction)

## How to test a pack

1. Download the pack you were asked to test.
2. Extract the zip.
3. Open `mission-dashboard.html` in your browser.
4. Read in Atlas.
5. Code in your own project.

If `mission-dashboard.html` opens in VS Code, open the extracted folder in Windows File Explorer and double-click the file there.

## For testers

You do not need to browse the folders unless you want to.

Start with the dashboard or reader file.

## Current test questions

Use one focused question per test round.

Examples:

- Does opening `mission-dashboard.html` feel like reading a clear guide instead of browsing project files?
- Is it clear that the Rookie Pack starts after your Vue project already exists?
- Do the Project Start Guides help you begin without feeling like a full tutorial?
- Does the Full Pack AI guidance help you stay in control during coding sessions?
