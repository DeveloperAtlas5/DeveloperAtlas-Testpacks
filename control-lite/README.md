# Atlas Control Lite

This is a local browser-based preview of the Atlas artifact workflow.

## Run it

1. Download or clone this repository.
2. Open `control-lite/index.html` in a modern browser.
3. Define one bounded change.
4. Generate and review the Change Contract, AI instruction, Evidence Record, and Decision Record.

The tool has no backend, telemetry, account, AI connection, or command execution. Generated files are
templates; a human remains responsible for scope, evidence, and the final decision.
