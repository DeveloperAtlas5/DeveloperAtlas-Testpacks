# Contributing to Developer Atlas Public

Developer Atlas is in public preview and is not yet accepting unrestricted
implementation contributions.

Useful contributions today include:

- reproducible bugs in a public pack or example;
- unclear terminology or first steps;
- broken public links;
- accessibility barriers;
- privacy or trust concerns without sensitive exploit details;
- a concrete AI-assisted workflow in which scope or understanding was lost.

## Reporting well

Include what you opened, what you expected, what happened, and the smallest safe
reproduction. Screenshots are useful only after removing private source, local
paths, account names, credentials, and personal information.

Do not submit proprietary project source, raw AI conversations, client names,
credentials, databases, tokens, or exploit details in a public issue.

## Pull requests

Please open an issue before a substantial pull request. Small corrections to
public documentation and runnable examples are welcome when they preserve the
human-control, privacy, and beginner-readability boundaries.
