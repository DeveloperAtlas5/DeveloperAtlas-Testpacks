# Rookie Pack

## Best for

Use this pack if:

- you already have a basic Vue project, or
- you are ready to work on a simple Vue to-do app, and
- you want one calm mission instead of a big tutorial.

## What you will do

You will make a Vue to-do app remember items after refresh.

The mission focuses on:

- finding the right file
- understanding the current to-do state
- saving to browser storage
- loading saved items after refresh
- checking your work

## Expected time

30–60 minutes.

## Start here

If this folder includes a zip:

1. Download or copy `developer-atlas-rookie-pack.zip`.
2. Extract it.
3. Open the extracted folder.
4. Open `START_HERE.md` or `mission-dashboard.html`.

## Important

This pack assumes your Vue project already exists.

If you do not know how to create or open a Vue project yet, start with:

```txt
packs/beginner-preview/
```

or wait for the full Beginner Pack.
