# Example — Park Scope Creep

## Situation

Current mission:

```txt
Make the todo list remember items after refresh.
```

AI suggests:

```txt
We could also add accounts, due dates, drag-and-drop, filters, and a backend.
```

## Problem

Those ideas might be useful, but they do not belong in the current mission.

## Better response

```txt
Use the Atlas AI action model.
Park the extra ideas.

Current mission:
Make the todo list remember items after refresh.

Do not build extra features.
Give me only the next small step.
```

## Good parked ideas

```txt
Later:
- accounts
- due dates
- filters
- drag-and-drop
- backend storage
```

## Lesson

Parking ideas keeps momentum without losing the current mission.
