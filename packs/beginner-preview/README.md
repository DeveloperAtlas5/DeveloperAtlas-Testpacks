# Beginner Preview

## Best for

Use this preview if:

- you are new to web development
- you are unsure how HTML, CSS, and JavaScript fit together
- you get stuck before the coding mission even starts
- opening files/projects feels confusing

## Current status

This is a preview path.

It is not a complete beginner course yet.

## What we are testing

We want to know whether the planned beginner path sounds clear and useful.

The future Beginner Pack should help with:

- what HTML is for
- what CSS is for
- what JavaScript is for
- how files connect
- how to open a project folder
- how to open an HTML page in the browser
- how to know where to type code

10–20 minutes for this preview.
