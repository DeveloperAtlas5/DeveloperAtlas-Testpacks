# Beginner web foundations

For people opening HTML, CSS, and JavaScript files for the first time.

- **Finish:** one readable page and one small interaction.
- **Time:** 2–4 short sessions; each mission is designed to stop cleanly.
- **Requirements:** a text editor and a modern browser.

## Start

1. [Mission 1 — readable HTML](../../content/missions/free/MISSION-HTML-001.md)
2. [Mission 2 — CSS](../../content/missions/en/MISSION-CSS-002.md)
3. [Mission 3 — JavaScript](../../content/missions/en/MISSION-JAVASCRIPT-003.md)
4. [Open the small runnable example](../../examples/html-css-js-starter/)

## Expected result

You can point to the HTML structure, explain what the stylesheet changes, and describe why the click
handler updates text. The goal is understanding, not visual polish.
