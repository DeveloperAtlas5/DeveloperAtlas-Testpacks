# Small Patch

**Level:** AI Collaboration

## What it means

A small patch changes the least amount of code needed to solve the current problem.

## Use it when

Use a small patch when you want code you can review, test, and undo.

## Tiny example

```txt
Patch only the smallest necessary change.
Show the filename first.
Do not add unrelated features.
```

## Watch out

A readable, testable patch is better for learning than a clever patch.
