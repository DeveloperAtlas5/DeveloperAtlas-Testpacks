# Browser vs Editor

**Level:** Beginner

## What it means

Your editor is where you change files. Your browser is where you test the result.

## Use it when

Use the editor to write code. Use the browser to check what the user sees.

## Tiny example

```txt
Editor: index.html, style.css, script.js
Browser: open index.html, click buttons, refresh page
```

## Watch out

If HTML opens in VS Code, right-click and choose Open with browser.
