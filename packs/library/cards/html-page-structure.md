# HTML Page Structure

**Level:** Beginner

## What it means

HTML is the structure of a web page. It tells the browser what elements exist.

## Use it when

Use it for headings, text, buttons, inputs, links, sections, and images.

## Tiny example

```txt
<h1>Todo List</h1>
<input placeholder="New todo">
<button>Add</button>
```

## Watch out

HTML does not decide the visual style or click behavior.
