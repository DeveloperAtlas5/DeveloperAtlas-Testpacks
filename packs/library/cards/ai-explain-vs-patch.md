# AI Explain vs Patch

**Level:** AI Collaboration

## What it means

Explain means the AI teaches first. Patch means the AI changes code.

## Use it when

Use Explain when the problem is unclear. Use Patch only when the change is small enough to review.

## Tiny example

```txt
Start with Explain.
Do not patch code yet.
Tell me what this error means and which file probably matters.
```

## Watch out

If you ask 'fix this' too early, AI may rewrite too much.
