# JavaScript Page Behavior

**Level:** Beginner

## What it means

JavaScript controls behavior. It reacts to clicks, changes text, validates input, stores data, and updates the page.

## Use it when

Use it when something on the page needs to respond or change.

## Tiny example

```txt
button.addEventListener('click', () => {
  console.log('Button clicked')
})
```

## Watch out

Keep functions small and names clear.
