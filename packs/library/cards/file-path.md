# File Path

**Level:** Beginner

## What it means

A file path tells you where a file lives inside folders.

## Use it when

Use file paths to find the exact file a mission or AI response is talking about.

## Tiny example

```txt
src/components/TodoItem.vue

src folder > components folder > TodoItem.vue file
```

## Watch out

Do not edit a similar-looking file just because the name is close.
