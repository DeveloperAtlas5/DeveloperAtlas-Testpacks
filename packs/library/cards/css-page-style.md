# CSS Page Style

**Level:** Beginner

## What it means

CSS controls how the page looks: spacing, colors, layout, size, borders, and responsive behavior.

## Use it when

Use it when the page works but looks plain or messy.

## Tiny example

```txt
button {
  padding: 8px 12px;
  border-radius: 6px;
}
```

## Watch out

Do not use CSS to store data or handle clicks.
