# Scope Creep

**Level:** AI Collaboration

## What it means

Scope creep happens when extra ideas enter the mission and distract from the original goal.

## Use it when

Use this when AI suggests features that sound useful but are not needed right now.

## Tiny example

```txt
Current goal: Make todos persist after refresh.

Park:
- accounts
- database
- drag-and-drop
- filters
```

## Watch out

Good ideas can still be wrong for the current mission.
