# Developer Atlas Library Preview

## Best for

Use this preview if:

- you want quick explanations of programming terms
- a mission uses a word you do not fully understand
- you want small cards instead of long documentation
- you are testing whether the card format feels useful

## What this preview contains

This preview includes cards for:

```txt
HTML
CSS
JavaScript
browser/editor basics
file paths
terminal commands
Vue
localStorage
AI collaboration
scope control
small patches
```

## Start here

Open:

```txt
cards/index.md
```

Then pick one card.

You do not need to read everything.
