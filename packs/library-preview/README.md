# Library Preview

## Best for

Use this preview if:

- programming words slow you down
- you want quick explanations without a full tutorial
- you want to understand terms before changing code
- you switch between tools like Vue, Laravel, PHP, SQL, HTML, CSS, or JavaScript

## Current status

This is a preview path.

The full Concept Library remains private for now.

## What the Library is for

The Library should answer questions like:

- What is `ref()` in Vue?
- What is `reactive()`?
- What are props?
- What is a route?
- What is a migration?
- When should AI explain before changing code?
