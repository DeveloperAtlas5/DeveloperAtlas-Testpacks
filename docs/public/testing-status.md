# Testing status

Atlas currently has automated validation, founder dogfooding, and formative
testing through a five-person embedded design cohort. One external-alpha round
has been preserved and has already driven packaging, search, progressive-
disclosure, terminology, and trust changes. A same-tester retest remains pending.

The private repository currently validates 54 maintained knowledge nodes, 664
Lexicon entries, 752 generated Compendium pages, and the Control, Continuity,
Navigator, Compendium, security, policy, and lifecycle test suites.

The public-repository workflow validates the export boundary and local links, then
runs the browser-list lifecycle in Chromium. Its regression coverage includes add,
complete, reload, remove, and HTML-like input rendered as literal text.

This supports technical consistency and formative usefulness. Public beta,
retention, willingness to pay, and broad independent validation have not been
established.
