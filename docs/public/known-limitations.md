# Known limitations

- The project is in internal alpha with a deliberately limited public preview.
- Independent node review is incomplete.
- Current flow support focuses on a limited set of web-development stacks.
- Broad accessibility, market, retention, and pricing validation are incomplete.
- The local browser handoff requires a running Compendium or hosted HTTPS target.
- Control Lite's editor-native lifecycle is implemented in development, but clean-profile VSIX
  validation and broader external workflow testing remain incomplete.
- Atlas Continuity still needs its recommended real Windows project validation.
- Product surfaces are broader than the currently proven repeated workflow.
