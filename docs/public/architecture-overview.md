# Architecture overview

Atlas Core combines Knowledge Nodes, steering patterns, guardrails, human
preferences, verified project context, and orchestration. Product surfaces consume
the same reviewed foundation rather than maintaining independent editorial copies.

```text
human goal
  -> scoped AI instruction
  -> traceable change
  -> understanding and impact
  -> verification evidence
  -> accept, revise, or reject
  -> remembered decision
```

The human owns the goal and final acceptance. Automated checks and AI claims are
never presented as independent human proof.
