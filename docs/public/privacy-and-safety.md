# Privacy and safety

Developer Atlas is designed around local-first processing, explicit actions,
and human acceptance.

## Current commitments

- The current Navigator preview has no telemetry, account, or payment system.
- Workspace source is not uploaded by Atlas Navigator.
- Atlas does not silently run AI agents or accept changes.
- Remote knowledge links require HTTPS; HTTP is restricted to localhost.
- Potentially risky workspace actions require an explicit trust decision.
- Public exports exclude private project material, personal data, credentials,
  raw feedback, databases, and raw AI conversations.

## Limits

These are product boundaries, not a third-party security certification. Preview
software may contain defects. Report suspected vulnerabilities privately through
GitHub Security Advisories rather than a public issue.
