# Developer Atlas Public Test Round 01

Thank you for testing Developer Atlas.

You are not being tested.<br>
The pack is being tested.

## Goal

We are testing whether people can:

```txt
choose one pack
open the right starting file
understand what to do next
make progress without overwhelm
give useful feedback
```

## Choose one pack

| If you are... | Start with... |
|---|---|
| New to web development | [`packs/beginner/`](../../../packs/beginner/) |
| Ready for one small Vue mission | [`packs/rookie/`](../../../packs/rookie/) |
| Looking for quick programming explanations | [`packs/library/`](../../../packs/library/) |
| Using AI coding tools | [`packs/ai-collaboration/`](../../../packs/ai-collaboration/) |

## Important

Please test **one pack only**.

Do not test the whole repo at once.

## What to report

Please tell us:

- what you opened first
- whether you knew what to do next
- what felt clear
- what felt confusing
- where you stopped
- whether it felt calm or overwhelming
- whether you would try another pack
