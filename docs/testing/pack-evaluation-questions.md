# Pack evaluation questions

These questions are for facilitated testing and research. They are intentionally separate from the
normal product and learning entry pages.

## Beginner

- Was the first file and first action obvious?
- Could the participant describe the expected result before starting?

## Rookie

- Could the participant complete add, toggle, remove, and reload without scope expansion?
- Did the safe text-rendering explanation change what they would write next time?

## Library

- Did the node answer the immediate concept question quickly enough to continue?
- Was the difference between automated review and independent review clear?

## AI Collaboration

- Could the participant distinguish Explain, Suggest, Patch, Confirm, Park, and Refuse?
- Did the Change Contract alter the scope or review behavior of the task?

The participant is not being graded. Record friction in the experience, not a score for the person.
