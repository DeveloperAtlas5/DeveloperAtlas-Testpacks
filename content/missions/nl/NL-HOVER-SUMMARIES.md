# Nederlandse hover summaries

These are short Dutch reminders for the extension.

Code keywords stay in English.

| Concept ID | English term | Dutch hover summary |
|---|---|---|
| CANON-HTML-001 | HTML | HTML geeft de structuur van een webpagina. |
| CANON-CSS-001 | CSS | CSS bepaalt hoe HTML eruitziet. |
| CANON-JAVASCRIPT-001 | JavaScript | JavaScript voegt gedrag toe aan een pagina. |
| CANON-DOM-001 | DOM | De DOM is de levende paginaversie die JavaScript kan lezen en aanpassen. |
| CANON-EVENT-001 | Event | Een event is iets dat gebeurt, zoals een klik. |
| CANON-LOCALSTORAGE-001 | localStorage | localStorage bewaart kleine browserdata na refresh. |
| SUPPORT-CLASS-001 | class | Een class is een herbruikbaar label voor HTML-elementen. |
| SUPPORT-BUTTON-001 | button | Een button is een knop waar een gebruiker op kan klikken. |
| SUPPORT-QUERYSELECTOR-001 | querySelector | querySelector laat JavaScript een element op de pagina vinden. |
| SUPPORT-ADDEVENTLISTENER-001 | addEventListener | addEventListener laat JavaScript wachten op een actie zoals een klik. |
