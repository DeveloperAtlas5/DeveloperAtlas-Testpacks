# MISSIE-HTML-001 — Je eerste menselijk leesbare webpagina

## Missiedoel

Maak een kleine HTML-pagina en leer hoe een browser een bestand leest.

## Waarom dit belangrijk is

HTML is de structuur van een webpagina.

In Developer Atlas leer je niet alleen dat code werkt. Je leert ook dat code leesbaar moet zijn voor mensen.

## Voor je begint

Maak een map:

```text
atlas-doorway/
```

Maak daarin een bestand:

```text
index.html
```

## Concepten

- bestand
- map
- HTML
- browser
- structuur
- inspringing

## Stap 1 — Maak de basis

Zet dit in `index.html`:

```html
<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>My First Atlas Page</title>
</head>
<body>
  <main>
    <h1>Hello, I am learning code</h1>

    <p>
      This is my first Developer Atlas page.
      I am learning how files connect.
    </p>
  </main>
</body>
</html>
```

## Controlepunt

Open `index.html` in je browser.

Je zou een titel en een stukje tekst moeten zien.

## Veelgemaakte fouten

| Probleem | Waarschijnlijke oorzaak | Oplossing |
|---|---|---|
| Browser toont niets | Bestand niet opgeslagen | Sla `index.html` op |
| Rare tekens | `meta charset` ontbreekt | Voeg `<meta charset="UTF-8">` toe |
| Structuur is rommelig | Geen inspringing | Gebruik consistente spaties |

## Herstelkaart

Als je vastloopt:

1. Stop met willekeurig aanpassen.
2. Sla het bestand op.
3. Open het juiste bestand.
4. Vergelijk je HTML met het voorbeeld.
5. Vraag AI eerst om uitleg, niet om een volledige rewrite.

## Veilige AI-prompt

```text
Leg uit wat er fout kan zijn in mijn HTML. Herschrijf niet mijn hele bestand. Geef alleen de kleinste fix.
```

## Human Preference check

- [ ] De structuur is duidelijk.
- [ ] De inspringing is consistent.
- [ ] De tekst is begrijpelijk.
- [ ] Je kunt uitleggen wat `<head>` en `<body>` doen.

## Bewijs dat je klaar bent

Je kunt uitleggen:

- wat `index.html` is
- waarom de browser het bestand kan openen
- wat HTML doet
- waarom inspringing helpt
